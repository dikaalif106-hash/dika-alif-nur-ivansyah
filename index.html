Activities

DNS Server Debian

Saya belajar membuat dan mengkonfigurasi DNS Server menggunakan Debian Linux. Dalam kegiatan ini saya mempelajari cara menghubungkan domain dengan alamat IP menggunakan layanan Bind9 sehingga client dapat mengakses server dengan lebih mudah menggunakan nama domain.

Web Server Debian

Saya membuat web server menggunakan Debian Linux dengan Apache2. Saya belajar cara meng-host website, mengatur virtual host, serta memastikan website dapat diakses dengan baik melalui jaringan lokal maupun internet.

SSH Server Debian

Saya mempelajari cara membuat SSH Server di Debian untuk melakukan remote access ke server secara aman. Dalam kegiatan ini saya belajar konfigurasi OpenSSH.

WordPress Debian

Saya belajar menginstall dan mengkonfigurasi WordPress di Debian menggunakan Apache, PHP, dan MySQL/MariaDB. Saya juga mempelajari cara membuat website dinamis dan mengelola tampilan website menggunakan WordPress.

Networking practice

Saya juga mempelajari dasar networking seperti konfigurasi IP address, subnetting, routing dasar, dan pengenalan perangkat jaringan menggunakan Mikrotik dan Cisco dasar.
